OVERVIEW ABOUT QUIZ

This Quiz helps you to test your knowledge on HTML and JAVA.
This quiz helps you to take simple mock tests on Java and Html,so that it helps you to get an idea on basic questions of Html and Java.
It contains two categories of tests. Each has 10 questions.

Below every question, we have two buttons Previous and Next which helps us to navigate
to previous question and next question.
After answering all the questions, result can be viewed by clicking on Show Result button
